# Minesweeper4java
A simple test driven minesweeper clone for java

## How to play (console view)

```sh
java -jar minesweeper4java-0.1.jar

Enter column count (positive integer): 15
open 5 5
open 4 5
mark 4 4
...
```

Enjoy!

**Feedback and pull requests appreciated**
